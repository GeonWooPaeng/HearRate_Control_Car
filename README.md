# HearRate_Control_Car
심장박동 센서(핸들)를 이용하여 큰 사고를 방지하는 차를 아두이노를 통해 만들었습니다.
(심장박둉 센서를 핸들에 두어 제어하는 방식입니다 / 특정 범위( < bpm 70)인 경우 모터와 비상등을 제어)
1. 모터 제어
2. 비상등 제어

- Arduino, C
